# 📊 Health Impact by Social Media - Data Analytics Project

## 📌 Project Overview
This project analyzes the impact of social media usage on students' mental and physical health using Python. The goal is to understand how factors like daily social media usage, sleep hours, screen time, and physical activity affect stress, anxiety, addiction, and depression.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing.
- Explore the dataset using Exploratory Data Analysis (EDA).
- Visualize patterns and relationships between variables.
- Generate meaningful insights from the data.

---

## 📂 Dataset Information
The dataset contains **1,200 student records** with **13 features**.

### Features
- Age
- Gender
- Daily Social Media Hours
- Platform Usage
- Sleep Hours
- Screen Time Before Sleep
- Academic Performance
- Physical Activity
- Social Interaction Level
- Stress Level
- Anxiety Level
- Addiction Level
- Depression Label

---

## 🔍 Data Cleaning
The following data quality checks were performed:

## 📊 Exploratory Data Analysis (EDA)

---

## 📈 Key Insights
- Dataset contains 1200 student records.
- Gender distribution is nearly balanced.
- Social media usage varies from 1 to 8 hours per day.
- Sleep hours, stress level, anxiety level, and addiction level show noticeable variation among students.
